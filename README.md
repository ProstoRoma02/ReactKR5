## 📋 Описание проекта

Это учебное приложение создано для демонстрации современных подходов к разработке на Vue 3. Проект включает в себя различные интерактивные компоненты, маршрутизацию, управление состоянием и адаптивный дизайн.

### Основные возможности

- ⚡ **Реактивность** - использование `ref`, `reactive` и `computed`
- 🧩 **Компоненты** - модульная архитектура с переиспользуемыми компонентами
- 🛣️ **Маршрутизация** - Vue Router 4 с динамическими маршрутами и вложенной навигацией
- 🎨 **Современный дизайн** - адаптивный интерфейс с градиентами и анимациями
- 📱 **Отзывчивость** - поддержка мобильных устройств и планшетов

## 🛠️ Технологии

- **Vue 3** - прогрессивный JavaScript фреймворк с Composition API
- **Vite** - быстрый инструмент сборки и разработки
- **Vue Router 4** - официальный роутер для Vue.js
- **CSS3** - современные стили с Flexbox и Grid
- **JavaScript ES6+** - современный синтаксис JavaScript

## 🎯 Функциональность

### Компоненты

1. **ReactiveForm** - Интерактивная форма с:
   - Валидацией полей в реальном времени
   - Различными типами input (text, email, select, radio, checkbox)
   - Предварительным просмотром данных

2. **ItemList** - Динамический список с:
   - Добавлением/удалением элементов
   - Фильтрацией и сортировкой
   - Редактированием элементов
   - Массовыми операциями

3. **Timer** - Многофункциональный таймер с:
   - Режимами секундомера и обратного отсчета
   - Промежуточными результатами (лапами)
   - Статистикой и уведомлениями

4. **SearchInput** - Поиск с автодополнением:
   - Кастомный v-model
   - Debounced поиск
   - Навигация с клавиатуры

5. **SlotDemo** - Демонстрация слотов:
   - Именованные слоты
   - Scoped слоты с передачей данных
   - Fallback контент

### Страницы

- **Главная** - Презентация проекта с анимированной статистикой
- **Демо** - Интерактивные примеры всех компонентов
- **Профиль** - Управление профилем пользователя с динамическими данными
- **Настройки** - Многоуровневые настройки с вложенной маршрутизацией
- **404** - Креативная страница ошибки с поиском и предложениями

## 🚀 Установка и запуск

### Предварительные требования

- Node.js (версия 18 или выше)
- npm или yarn

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/vue-interactive-app.git
cd vue-interactive-app
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите сервер разработки:
```bash
npm run dev
```

4. Откройте браузер и перейдите по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

### Деплой на GitHub Pages

```bash
npm run deploy
```
